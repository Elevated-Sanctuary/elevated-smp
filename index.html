Installation guide

1. Install Fabric

Download and install fabric

  • Select Minecraft version
  • Check the "Create Profile" box
Download Fabric
2. Download the mods

Mods are provided below

  • Win + R %appdata% .minecraft
  • Locate "mods" folder, if there's none, create one
  • Unzip and place the downloaded mods into the "mods" folder
Download Mods
3. Configure the game

Increase allocated memory in the Installation tab

  • Open Minecraft Launcher
  • Installation tab
  • Select the installed Fabric profile
  • More options JVM Arguments
  • Change -Xmx2G to a higher value based on your system's RAM e.g., -Xmx8G for 16GB RAM (30-70% recommended)
4. Launch the game

Increase allocated memory in the Installation tab

  • Open Minecraft Launcher
  • Change profile on the bottom left
  • Select the installed Fabric profile
  • Click "Play"
